Para subir cambios:
1.- git add .
2.- git commit -m "comentario" ("" si va en el comando)
3.- git push origin master
4.- listo :D

Para bajar cambios:
1.- Git fetch origin
4.- git pull origin
5.- Listo :D

Para bajar el repositorio por primera vez:
1.- git clone 'direccion web del repositorio'.git ('' no van en el comando)
